# 教师成绩分析工具

一个基于 **Streamlit** 开发的教师辅助工具，用于上传 Excel 成绩表后，快速完成班级、学科等维度的成绩统计与分析。

这个项目主要想解决老师日常处理成绩时反复筛选、计算、整理数据比较麻烦的问题，让 Excel 成绩表可以更直观地展示分析结果。

---

## 功能介绍

目前已实现：

- 上传 Excel 成绩表
- 自动识别常见的班级列、姓名列、考号列和学科成绩列
- 支持按班级筛选学生成绩
- 支持班级与科目联动分析
- 查看各学科成绩数据
- 统计平均分、最高分、最低分等基础数据
- 统计及格率、优秀率等成绩情况
- 对成绩数据进行可视化展示
- 支持导出部分分析结果
- 对部分格式不规范的数据提供提示，避免页面直接报错

---

## 使用场景

适用于以下情况：

- 教师上传一次考试成绩表后，快速查看整体成绩情况
- 单独查看某个班级的成绩表现
- 对某一门学科进行成绩分析
- 快速统计优秀率、及格率等数据
- 减少手动筛选 Excel、重复计算成绩的时间

## 使用方式

### 1. 下载项目

将项目下载或克隆到本地：

```bash
git clone 你的项目地址
````

也可以直接在 GitHub 点击 `Code` → `Download ZIP` 下载。

---

### 2. 安装依赖

打开终端，进入项目文件夹后运行：

```bash
pip install -r requirements.txt
```

项目主要依赖：

```text
streamlit
pandas
openpyxl
```

---

### 3. 启动项目

在项目根目录运行：

```bash
streamlit run app.py
```

启动后，终端会显示本地访问地址，通常为：

```text
http://localhost:8501
```

复制到浏览器打开即可使用。

## Excel 表格格式说明

建议上传 `.xlsx` 格式的成绩表。

表格中建议包含以下信息：

| 班级   | 考号      | 姓名 | 语文 | 数学 | 英语 | 物理 | 历史 | 政治 |
| ---- | ------- | -- | -- | -- | -- | -- | -- | -- |
| 2401 | 2401101 | 张三 | 88 | 96 | 92 | 85 | 78 | 90 |

程序会尝试自动识别：

* 班级列，例如：`班级`、`班号`、`班别`
* 姓名列，例如：`姓名`、`学生姓名`
* 考号列，例如：`考号`、`学号`
* 学科成绩列，例如：`语文`、`数学`、`英语` 等

建议确保：

* 第一行是表头；
* 成绩列尽量为数字；
* 不要在表格开头加入太多合并标题行；
* 上传前确认当前 Sheet 是需要分析的成绩表。

> 目前版本默认读取 Excel 的第一个工作表。后续会增加“选择工作表”的功能。
