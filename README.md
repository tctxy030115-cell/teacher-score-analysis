# 教师成绩分析工具

一个面向教师的智能成绩分析工具。

支持 Excel 成绩表导入，自动完成成绩统计、等级划分、成绩分布分析，并生成可直接用于教学总结的 Word 成绩分析报告。

项目目标：

> 帮助教师减少重复的数据整理工作，把更多时间用于学生分析和教学调整。


## ✨ 项目介绍

日常教学中，考试结束后教师通常需要：

- 整理 Excel 成绩表；
- 计算平均分、最高分、最低分；
- 统计优秀率、及格率；
- 分析成绩分布；
- 筛选优秀学生和待提升学生；
- 制作考试分析报告。

这些工作重复且耗时。

本项目尝试通过 Python + Streamlit 将这一流程自动化：
Excel成绩表
↓
自动识别字段
↓
成绩统计分析
↓
可视化图表
↓
学生分层分析
↓
自动生成 Word 报告

让教师从“整理数据”转向“分析数据”。

# 🚀 当前功能


## 1. Excel 成绩导入

支持常见教师成绩表结构：

- Excel 文件上传；
- 工作表选择；
- 成绩列自动识别；
- 学生姓名识别；
- 班级信息识别。


支持：

- 单科成绩分析；
- 多班级成绩比较；
- 无班级列数据分析。


---

## 2. 单班成绩分析


支持查看：

### 核心指标

- 参考人数
- 平均分
- 最高分
- 最低分
- 及格率
- 优秀率


### 成绩分布

自动生成：

- 分数段分布图；
- 等级结构分析；
- 成绩集中区间判断。


### 学生分层

自动生成：

- 优秀学生名单；
- 待提升学生名单。


支持：

- 指定班级分析；
- 全部学生分析。


---

## 3. 多班级横向比较


支持多个班级同时选择：

分析：

- 平均得分率；
- 及格率；
- 优秀率；
- 等级结构。


帮助教师快速了解班级之间差异。


---

## 4. 自动生成 Word 成绩分析报告 ⭐


项目目前重点功能。

系统可以自动生成专业 Word 报告：

包含：

### 第1页
考试信息与报告说明

### 第2页
核心指标与总体分析

### 第3页
成绩分布分析

### 第4页
等级结构与各科表现

### 第5页
优秀学生、待提升学生和教学建议


报告特点：

- A4 排版；
- 自动插入图表；
- 自动生成分析文字；
- 支持全部学生和具体班级；
- 名单自动控制长度。


示例流程：
上传成绩Excel

↓

点击生成报告

↓

得到 Word 成绩分析报告
